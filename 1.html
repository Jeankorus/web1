HTML

HyperText Markup Language (HTML) is the standard markup language for creating web pages and web applications.

This format description is for HTML 5, standardized in two coordinated efforts. One is by the Web Hypertext Application Technology Working Group (WHATWG), which maintains a specification for HTML as a modularized "living standard" at https://html.spec.whatwg.org/.
